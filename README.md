# 敲磚塊遊戲

這是一個使用 Python 開發的簡單敲磚塊(打磚塊)遊戲專案。

## 專案結構

- `main.py` - 遊戲主要程式進入點。

## 執行方式

1. 建議建立虛擬環境並安裝相依套件（如果有 requirements.txt）：

```powershell
python -m venv .venv; .\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

2. 直接執行遊戲：

```powershell
python main.py
```
